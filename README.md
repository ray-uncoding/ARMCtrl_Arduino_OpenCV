# 📷🔌 ARMCtrl_Arduino_OpenCV

一個使用 **OpenCV + Python** 搭配 **Arduino + 繼電器控制機器手臂** 的影像辨識專案！  

電腦能根據鏡頭畫面中的顏色與形狀，透過arduino繼電器模組輸出對應的數位訊號給機器手臂IO，並使機器手臂執行對應的動作。
